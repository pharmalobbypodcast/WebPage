# 药厂门口 多页面静态网站 Mockup

这个文件夹可以直接用于 GitHub Pages。

## 文件说明
- `index.html`：首页
- `about.html`：关于我们
- `episodes.html`：节目库
- `guest.html`：嘉宾合作
- `ask.html`：匿名提问
- `contact.html`：联系我们
- `styles.css`：统一样式

## 如何本地预览
直接双击 `index.html` 用浏览器打开即可。

## 如何部署到 GitHub Pages
1. 登录 GitHub
2. 新建一个 repository，例如 `yaochangmenkou-site`
3. 把以上文件上传到仓库根目录
4. 打开仓库的 Settings → Pages
5. Source 选择 Deploy from a branch
6. Branch 选择 main，folder 选择 /root
7. 保存
8. 等几十秒后会生成一个网址，例如：
   `https://你的用户名.github.io/yaochangmenkou-site/`

## 下一步你只需要替换这些内容
- 主持人真实介绍
- 节目真实标题和链接
- 平台链接
- 合作邮箱 / 微信助手
- Tally 匿名表单 embed code（放在 `ask.html`）

## 小提示
第一版先不要追求太复杂。
能上线、能发给嘉宾、能给听众匿名提问，就已经很有价值了。
