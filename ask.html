匿名提问

这是你们网站里很有辨识度的一页。它不需要复杂论坛，第一版只要把“匿名提问箱”做好，就已经很有社区感了。

这页为什么重要

社交平台里大家往往带着身份、头像和关系网络，很多问题不方便公开问。匿名提问页会让更多真实困惑浮出来，这会成为你们非常有价值的内容来源。

  • 更适合问职业焦虑、转型犹豫、求职困惑
  • 更容易收到真实而具体的问题
  • 可以反向变成节目选题或精选问答内容

Tally 嵌入位（后续替换）

你以后只要把 Tally 的 embed code 粘到这里，就能上线匿名表单。

<iframe data-tally-src="https://tally.so/r/your-form-id" loading="lazy" width="100%" height="600" frameborder="0" marginheight="0" marginwidth="0" title="匿名提问"></iframe>

建议表单里放哪些问题

先保持简单，问题越少，提交率越高。

Q1

你的问题是什么？

开放文本框，允许对方自由表达。

Q2

问题分类

例如：求职 / 职场困惑 / 转行 / 行业观察 / 情绪压力。

Q3

是否同意匿名引用

可勾选“允许在节目或文章中匿名引用”。
